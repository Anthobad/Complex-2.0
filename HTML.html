Lesson 1: What is HTML and how do we use it?

The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets(CSS) and scripting languages such as JavaScript(Js).

HTML Tags are divided into 2 parts:

  1. Container
  2. Empty

Create an HTML code:

  1. Open Notepad(TextEdit on the Mac) and save as My First Webpage.htm
  2. Usually a page is divided into sections: Header, Body and Footer. HTML coding follow the same idea, but with no Footer. The later you creat it in the Body.
  3. Type the following: 

<HTML>
<HEAD>
     <TITLE>My First Webpage</TITLE>
</HEAD>
<BODY>
     This is my page in which you will find my personal information...
</BODY>
</HTML>

You can add within the Head and the Body parts what you need to insert in your page. For example, type under <HEAD> your title, as you can see above.

Save and refresh. You will see the title you added in the title bar, and the other in the Body part.
